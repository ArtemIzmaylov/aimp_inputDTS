About
===========================================
aimp_InputDTS - Free DTS Decoder Plugin for AIMP3 v3.60, written on Delphi XE7 and released under GNU GPL terms. 
Source code of the plugin you can find on https://github.com/ArtemIzmaylov/aimp_inputDTS

Based on libdca.
See http://www.videolan.org/developers/libdca.html for more information and updates.

Installation
===========================================
1. Move the aimp_InputDTS folder to the "<AIMP>\Plugins\" directory
2. Restart the Player
